3D Garbage Pro

  • Project Goal: Develop a model that could do 3D garbage image classification.
  • Tech Stack: Python, Tensorflow, CNN, GMM
  • Devpost URL: Link

The main goal of this project is to project 3D images to spheres with GMM and use a Spherical CNNs-based model to do 3D image classifications.

We developed an Image-based method that converts the 3D objects to spheres with a new way of sphere projections using Gaussian Mixture Model(GMM) and apply the inputs with the Spherical CNNs.
